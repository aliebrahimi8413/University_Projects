# شبیه‌ساز گیت مترو

یه گیت مترو با کارت‌خوان، سنسور، و درب که خودکار باز و بسته میشه. مسافر
میاد جلو، کارتش به‌طور خودکار چک میشه: اگه پول کافی داشته باشه درب باز
میشه و رد میشه، اگه نداشته باشه رد میشه (به‌معنی: قبولش نمی‌کنه) و برمی‌گرده.

## اجرا

```
python3 main.py
```

فقط به tkinter نیاز داره.

## فایل‌ها

- `card.py` -> کلاس Card (کارت مترو، موجودی داره)
- `passenger.py` -> کلاس Passenger (مسافر، یه کارت داره و یه وضعیت)
- `sensor.py` -> کلاس Sensor (تشخیص میده مسافر به یه نقطه‌ی خاص رسیده یا نه)
- `motor.py` -> کلاس Motor (درب رو کم‌کم باز و بسته می‌کنه، نه یهویی)
- `gate.py` -> کلاس Gate (مغز اصلی: تصمیم می‌گیره کارت قبول بشه یا نه، صف مسافرا رو مدیریت می‌کنه)
- `gui.py` + `main.py` -> رابط گرافیکی

## گیت چطور کار می‌کنه

هر بار فقط یه مسافر «فعاله» و بقیه تو صف منتظرن (ساده‌سازی کردم که همه
با هم قاطی نشن). مسافر فعال:

1. راه میره سمت گیت
2. وقتی به سنسور ورودی می‌رسه، کارتش خودکار چک میشه
3. اگه پول کافی داشت: پول کم میشه، درب باز میشه، رد میشه
4. اگه پول کافی نداشت: قرمز میشه و برمی‌گرده عقب (رد شده)
5. وقتی از سنسور خروجی رد شد، بعد از یه‌کم مکث درب بسته میشه

## دو باگ واقعی که موقع تست پیدا کردم

**باگ ۱:** اولین نسخه‌ای که نوشتم، مسافر بعد از رد شدن هنوز حرکتش وابسته
به این بود که درب باز باشه یا نه. چون درب طبق تایمر یه‌جایی وسط راه
(بین سنسور خروجی و لبه‌ی صحنه) شروع به بسته شدن می‌کرد، مسافر همون‌جا وامی‌ایستاد
و دیگه هیچ‌وقت حرکت نمی‌کرد - برای همیشه گیر می‌کرد! نتیجه: نفر بعدی صف
هم هیچ‌وقت نوبتش نمی‌رسید. با تست واقعی (اجرای ۴۰۰ تیک و دیدن این‌که
مسافر هنوز "passing" مونده) این رو پیدا کردم. راه‌حل: بعد از این‌که
مسافر از سنسور خروجی رد شد، دیگه حرکتش به وضعیت درب بستگی نداره (چون
واقعا دیگه از درب رد شده و آزاده).

**باگ ۲:** (این یکی تو پروژه‌ی قبلی پاندول پیدا شده بود، ولی همون درسو
اینجا هم رعایت کردم): مطمئن شدم که موقع اضافه شدن مسافر جدید، هیچ حالت
قدیمی از مسافر قبلی (مثل سنسورهای هنوز-فعال) باقی نمونه؛ برای همین توی
`_clear_active` هم `entry_sensor` و هم `exit_sensor` رو ریست می‌کنم.

## چیزایی که میشه اضافه کرد

- چند صف/چند گیت هم‌زمان (الان فقط یه گیت با یه صف داریم)
- شارژ کردن کارت وسط بازی (الان اگه پول نداشته باشه فقط رد میشه)
- صدا وقتی درب باز/بسته میشه یا کارت رد میشه
